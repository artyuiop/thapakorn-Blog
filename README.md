##  Blog
** REGISTRATION สมัครสมาชิก
* กรอกชื่อจริง-นามสกุล
* สร้างบัญชีชื่อผู้ใช้
* สร้างรหัสผ่าน
* เมื่อกรอกครบแล้ว ให้คลิกยืนยันการสมัคร จากนั้นคลิกเข้าสู่ระบบ

** LOGIN เข้าสู่ระบบ
* กรอกชื่อบัญชีผู้ใช้
* กรอกรหัสผ่าน
* คลิกปุ่มเข้าสู่ระบบ เพื่อเข้าสู่หน้าแรกของblog
* หากไม่มีบัญชีผู้ใช้ ให้คลิกสมัครสมาชิก

** HOME หน้าแรก
* สามารถอ่านบทความ ของผู้ใช้คนอื่นๆที่โพสบทความลงในblogได้
* สามารถคลิกปุ่มอ่านต่อที่มุมขวามือด้านล่าง เพื่ออ่านบทความ
* ไม่สามารถคอมเม้นใต้บทความของผู้ใช้อื่นได้ สามารถอ่านได้อย่างเดียว
* เมื่ออ่านจบ คลิกปุ่มกลับหน้าหลักที่มุมซ้ายมือด้านล่าง
* หากต้องการสร้างบทความลง blog ให้คลิกที่ manager 

** MANAGER ผู้จัดการ
* ระบุหัวข้อของบทความ
* ระบุรายระเอียดเนื้อหาของทบความที่ต้องการ
* เมื่อระบุหัวข้อและเนื้อหาของบทความเสร็จแล้ว สามารถคลิกปุ่มบันทึกข้อมูลที่มุมขวามือด้านล่าง เพื่อบันทึกบทความ
* เมื่อบันทึกแล้ว คลิกปุ่มกลับหน้าหลักที่มุมซ้ายมือด้านบนเพื่อดูบทความที่โพสลง blog

** Edit/Delete แก้ไข/ลบ รายการบทความของฉัน

** Edit/แก้ไข
* ถัดมาจากการสร้างบทความ เมื่อเลื่อนลงมาด้านล้างจะพบบทความทั้งหมดของผู้ใช้เองที่สร้างทั้งหมด
* หากต้องการแก้ไขบทความ ให้คลิกปุ่มสีเหลืองรูปไอคอนดินสอที่ด้านขวามือเพื่อแก้ไขบทความ
* เมื่อคลิกเข้ามาแล้วผู้ใช้สามารถแก้ไขได้ทั้งหมด ทั้งหัวข้อและรายระเอียดเนื้อหา
* เมื่อแก้ไขแล้วให้คลิกปุ่มบันทึกการแก้ไขที่ขวามือด้านล่าง เมื่อบันทึกการแก้ไขแล้วจะกลับมาที่หน้า manager
* หากไม่อยากแก้ไข สามารถคลิกปุ่มยกเลิกที่ด้านซ้ายล้าง เพื่อกลับไปหน้า manager
  
** Delete/ลบ
* หากต้องการลบบทความ ให้คลิกปุ่มสีแดงรูปไอคอนถังขยะที่ด้านขวามือถัดจากปุ่มแก้ไขเพื่อลบบทความ
* เมื่อคลิกแล้ว จะมีแจ้งเตือนเพื่อให้ผู้ใช้ยืนยันว่าต้องการที่จะลบบทความจริงหรือไม่
* หากต้องการ สามารถคลิก ตกลง/ok แล้วบทความถูกลบ
* หากลบบทความแล้ว ไม่สามารถกู้คืนบทความได้
* หากไม่ต้องการลบ สามารถคลิก ยกเลิก/canael แล้วบทความยังคงเดิม
* เมื่อผู้ใช้ แก้ไข บทความแล้ว สามารถคลิกกลับหน้าหลักที่มุมซ้ายมือด้านบน เพื่อดูบทความที่แก้ไขแล้ว
* เมื่อผู้ใช้ ลบ บทความแล้ว สามารถคลิกกลับหน้าหลักที่มุมซ้ายมือด้านบน เพื่อดูว่าบทความที่ลบยังอยู่หรือไม่

** LOGOUT ออกจากระบบ
* คลิกปุ่ม Logout ที่มุมขวาามือด้านบนสุด เพื่ออกจากระบบ
